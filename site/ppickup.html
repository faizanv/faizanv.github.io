Pebble Pickup is a Pebble Smartwatch app that won the Strap sponsor prize at MHacks V. It gives the user pickup lines to use at the flick of wrist.

The pickup lines were originally crowd-sourced through Twitter using #PebblePickup but I have changed the functionality to pull from an Express API I made instead to increase efficiency and better monitor content.
