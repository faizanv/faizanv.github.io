Get To A Better State Analysis of communication sentiment

Project Description

Get To A Better State was a project I made for the State Farm Hack Day while I interned there in Summer 2015. The idea is that it reads a user's sent email (with their permission of course) and runs it through a sentiment analysis engine and return a score from 1-100 based on how positive their communication is. It then lays it out similar to a fitness app to show how well the user is doing for the day/month/year.

Project Details
