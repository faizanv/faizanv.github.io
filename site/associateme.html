AssociateMe is a game that I developed with my team at HackFSU 2016 with the goal of using user input to collect data on associative networks; essentially getting data on people's though processes.

We aggregated all this data into a giant weighted graph visualization that shows actual patterns in how people think. If we had more time, we wanted to then use this graph data to procedurally generate new levels for the game and in doing so, collect more data.
