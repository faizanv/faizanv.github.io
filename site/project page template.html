dished Android Application

Project Description

Check It Out Here

Dished was an android app that I made with a team at HackGTeenee inNovember 2014 which won the event's Small Business Award. It serves as a rating app for restaurant menus. As as user searches up a restaurant on the app, it's menu is pulled up. The user can then see the different ratings for the menu items and add their own ratings on to that and it will update in real time. Unfortunately it is not available to download we just made it for educational purposes.
