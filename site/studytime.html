Study-Time is a flashcard management system that I made at HackIllinois February 2015. The system consists of a web app in which the user can add or delete flashcards that are stored on a database. Then the user can view these flashcards on a provided Pebble application.

Project Details

  • Pebble.JS
  • Node.JS
  • MongoDB
  • JQuery
